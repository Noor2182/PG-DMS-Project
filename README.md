# PG-DMS-Project

## Overview
An automated Document Management System (DMS) for **PG Studios** to manage large-scale NAS storage, focusing on smart file indexing and AI-driven workflows.

## Core Features
* **NAS Indexing & Search:** Scanning and organizing files for quick retrieval.
* **Smart File Management:** Automated renaming and project template creation.
* **AI Command Center:** Executing tasks via natural language.

## Tech Stack
* **Backend:** Python 
* **Frontend:** Streamlit 
* **Database:** SQLite 
* **AI:** OpenAI API integration
